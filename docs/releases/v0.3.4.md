# FieldLedger v0.3.4

Header and monochrome UI patch for device testing.

## Included

- Header chrome now uses a solid pinned surface so the `Command overview` bar stays visually fixed while content scrolls
- UI palette shifted to normal black and white tones instead of the warmer tinted scheme
- Light and dark mode labels simplified to `Light` and `Dark`

## Testing Notes

- This release asset is a debug APK for manual device testing
- Core app behavior is unchanged; this patch is focused on chrome and color adjustments

## APK

- File: `fieldledger-v0.3.4-debug.apk`
- Package: `com.indie.shiftledger`
