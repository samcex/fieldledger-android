# FieldLedger v0.3.7

Simpler job flow and currency update for device testing.

## Included

- The app now uses plainer labels and simpler wording across onboarding, home, jobs, and settings
- The new job flow shows the basic fields first and keeps address, notes, extra amounts, due date, and reminder details optional
- Work date, due date, and reminder date now use calendar pickers instead of manual date typing
- CHF was added as a supported currency option

## Testing Notes

- This release asset is a debug APK for manual device testing
- Core app behavior is unchanged; this patch is focused on simplifying the user experience

- File: `fieldledger-v0.3.7-debug.apk`
- Package: `com.indie.shiftledger`
