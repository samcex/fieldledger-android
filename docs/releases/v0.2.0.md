# FieldLedger v0.2.0

Second public MVP build of FieldLedger.

## Included

- Modernized Compose shell with refreshed colors, typography, and floating navigation
- Persistent currency settings for dashboard, capture flow, and job history
- First-launch onboarding with default currency selection
- Refreshed launcher icon and branded Android resources
- Play Store listing copy, screenshot brief, and SVG source assets in `docs/play-store/`
- Existing Room persistence and Google Play Billing scaffolding

## Testing Notes

- This release asset is a debug APK intended for manual device testing
- It is not a production-signed Play Store build
- Billing flows still use local client-side scaffolding in this version

## Known Gaps

- No cloud sync or backup
- No invoice export yet
- No reminder workflow yet
- Limited validation and no edit flow for saved jobs yet

- File: `app-debug.apk`
- Package: `com.indie.shiftledger`
