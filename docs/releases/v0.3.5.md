# FieldLedger v0.3.5

Readability and header cleanup patch for device testing.

## Included

- AMOLED mode now uses darker hero surfaces with stronger contrast so header and summary text stay readable
- Secondary dark-mode text has been brightened to avoid washed-out labels and support copy
- Duplicate scrolling labels were removed from the Pro and Settings hero sections so the fixed app header is the only page title

## Testing Notes

- This release asset is a debug APK for manual device testing
- Core app behavior is unchanged; this patch is focused on contrast and title presentation

- File: `fieldledger-v0.3.5-debug.apk`
- Package: `com.indie.shiftledger`
