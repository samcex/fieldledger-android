# FieldLedger v0.3.10

Job editing, clearer job actions, and updated invoice branding rules for device testing.

## Included

- Saved jobs can now be opened, edited, and updated directly from the Jobs screen
- Job action pills now wrap so `Edit` and `Delete` stay visible on smaller devices
- Company name remains free, while custom logo branding is now a Pro-only invoice feature
- PDF invoices now give the company logo more space in the header for Pro users

## Testing Notes

- This release asset is a debug APK for manual device testing
- Free users should still be able to set a company name and send PDFs, but logo branding should route through Pro

- File: `fieldledger-v0.3.10-debug.apk`
- Package: `com.indie.shiftledger`
