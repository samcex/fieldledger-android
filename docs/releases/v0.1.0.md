# FieldLedger v0.1.0

First public MVP build of FieldLedger.

## Included

- Dashboard with weekly billed revenue, estimated profit, costs, and follow-up count
- Job entry form with customer, service, costs, invoice stage, and live preview
- Job pipeline screen with saved jobs and invoice stage badges
- Local Room persistence
- Google Play Billing scaffold for future `Pro` subscriptions

## Testing Notes

- This release asset is a debug APK intended for manual device testing
- It is not a production-signed Play Store build
- Billing flows depend on Play product setup and are not ready for real purchases

## Known Gaps

- No invoice PDF/export yet
- No backend purchase verification
- No cloud sync or backup
- Limited validation and no onboarding flow yet
