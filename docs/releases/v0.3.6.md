# FieldLedger v0.3.6

Hide-on-scroll header update for device testing.

## Included

- The top chrome pill now appears only when a tab is at the top of its list instead of staying over the content while you scroll
- Dashboard, Capture, Ledger, Pro, and Settings now use tab-specific list state so the header visibility follows each screen correctly
- The header no longer blocks other pills or content when you scroll to the bottom of a page

## Testing Notes

- This release asset is a debug APK for manual device testing
- Core app behavior is unchanged; this patch is focused on header visibility and scroll behavior

- File: `fieldledger-v0.3.6-debug.apk`
- Package: `com.indie.shiftledger`
