# FieldLedger v0.3.3

Classic UI release for manual device testing.

## Included

- Full visual redesign across onboarding, dashboard, job capture, history, Pro, and settings
- New classic palette built around parchment, navy, brass, and oxblood accents
- Reworked app chrome with a new header and bottom navigation shell
- New shared Compose design primitives for cards, hero panels, pills, and metric tiles
- Renamed theme modes to `Classic Light` and `Night Ledger`

## Testing Notes

- This release asset is a debug APK for manual device testing
- Core behavior is unchanged; this release is focused on the UI overhaul
- Billing and backend verification behavior are unchanged from the prior release

- File: `app-debug.apk`
