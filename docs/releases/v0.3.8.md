# FieldLedger v0.3.8

Compact currency picker and invoice actions update for device testing.

## Included

- Currency selection now uses a compact dropdown picker in onboarding and settings instead of taking over the screen
- Jobs can now be marked paid directly from the Jobs screen
- Invoice sending is clearer in the Jobs screen with a `Send PDF` action
- Sending the PDF now marks the job as `Sent`, and the generated file is labeled as an invoice

## Testing Notes

- This release asset is a debug APK for manual device testing
- Core app behavior is unchanged; this patch is focused on reducing UI clutter and improving job actions

## APK

- File: `fieldledger-v0.3.8-debug.apk`
- Package: `com.indie.shiftledger`
