# FieldLedger v0.3.0

Third public MVP build of FieldLedger.

## Included

- PDF invoice summary export and Android share flow
- Due dates plus one-tap unpaid reminder scheduling and clearing
- Local reminder notifications with boot-time rescheduling
- Backend-verified Google Play Billing client
- Bundled Node verifier service in `backend/`
- Real release-signing support via local env or Gradle properties
- Updated release build that produces a signed `.aab`

## Testing Notes

- GitHub release assets can include both a debug APK and a release `.aab`
- `Pro` entitlements now require a reachable backend verifier URL
- The bundled backend is not deployed for you

## Known Gaps

- No deployed production backend environment is included
- No cloud sync or backup
- No edit flow for saved jobs yet

## Assets

- `app-debug.apk`
- `app-release.aab`
