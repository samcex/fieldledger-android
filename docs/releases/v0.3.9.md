# FieldLedger v0.3.9

Fixed-price jobs, branded invoice PDFs, and launcher polish for device testing.

## Included

- Jobs can now be charged as either hourly work or a single fixed price
- Invoice PDFs can now use a saved company name and optional logo from Settings
- The job action pills and paid status labels were tightened so short labels stay centered and readable
- The launcher icon has been redesigned with a cleaner ledger-and-checkmark mark

## Testing Notes

- This release asset is a debug APK for manual device testing
- Existing saved jobs should migrate forward to the new pricing schema automatically

## APK

- File: `fieldledger-v0.3.9-debug.apk`
- Package: `com.indie.shiftledger`
