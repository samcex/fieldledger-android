# FieldLedger v0.3.1

Patch release focused on the in-app UI and UX refresh.

## Included

- Warmer visual system with revised palette and typography
- Lighter app chrome and cleaner navigation shell
- Reworked dashboard hierarchy with stronger money/pipeline emphasis
- Faster-feeling job capture flow with the live draft summary surfaced earlier
- Cleaner jobs screen with better scanning for open work and reminder actions
- Refreshed onboarding, settings, and paywall presentation

## Testing Notes

- This release asset is a debug APK for manual device testing
- The redesign changes presentation only; billing/backend behavior is unchanged
